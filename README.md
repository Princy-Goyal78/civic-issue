## About the Project
A beginner-friendly civic issue reporting app created as part of learning frontend development using React and TypeScript.

##  Project Overview
IC Civic Connect is a web-based platform designed to help citizens connect with local civic services more efficiently. The platform allows users to raise civic issues and stay informed about community-related updates.

##  Features
- User-friendly interface for reporting civic issues  
- Centralized platform for civic communication  
- Simple and clean UI design  

##  Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend
